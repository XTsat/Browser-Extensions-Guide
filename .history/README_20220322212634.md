# 浏览器插件指南

## 什么是浏览器插件

浏览器插件也被称为浏览器附加组件（Browser-Extensions），是一组可以添加到浏览器中以增强其功能的软件组件。

浏览器插件就是可定制浏览体验的小型软件程序，让用户可以根据个人需要或偏好来定制浏览器。简而言之，插件就是可以让你的浏览器用起来更加方便顺手的工具。

插件之于浏览器就像 app 之于手机，exe 软件之于 windows 一样，有了插件的浏览器可以发挥出更多更好的功能。
